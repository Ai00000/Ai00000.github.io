# Ep.2 Async 异步

Async 异步的概念是微软基于多线程提出的 Task 解决方案，相比传统多线程具有很多优势。

  • ❤️相比协程来说，异步的方式不需要搭建基础的协程框架（ 不需要新创建一个方法 ），可以直接在方法中使用关键字实现等待。
  • 不需要启动
