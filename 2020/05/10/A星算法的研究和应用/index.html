# 算法定义

A* 算法常被表示为 f (n)=g (n)+h (n)

**f (n)** 表示从状态 n 到达目标状态的花费的预测值
**g (n)** 表示从初始状态到达状态 n 的花费
**h (n)** 表示从状态 n 到达目标状态的花费预测值（最佳路径）
