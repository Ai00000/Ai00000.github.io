快速开始

首先你需要一个已经备案的域名。其次你需要创建一个七牛云账号并绑定一个加速域名。

创建加速域名

创建好一个空间，点击绑定加速域名，输入一个二级地址。

在域名服务商处开启CAME

处理

为了方便大家上传图片，我这里使用 2 种不同的方案，大家选择其一即可。

下载MPic

MPic

下载MarkdownPicPicker

MarkdownPicPicker 是一个 md 写作工具，用于上传图片。
下载程式
下载后创建一个 config 文件夹，并将 config.ini 配置好再放进去。

config.ini 示例：

[basic]
picture_folder = pic #必需，用来在本地保存图片的副本，名字任意
picture_suffix = png #必需，图片后缀名，请使用png
# 图片上传插件的文件名，不带".py"
picture_host = QiniuUploader

[QiniuUploader]
access_key = Q6sS422O0fasfsassadasdfahqasdftqvyQasdf5Zvzw
secret_key = 6QtA121TxoSxZadffsdfasdfaaffasCmoOaB2aLObM
container_name = picturebed
url = http://7sbpmp.com1.z0.glb.clouddn.com/{}

注意：在实际使用中，请将上面的注释部分去掉，避免出现不必要的问题。

如果你希望只将图片的 url 复制到剪贴板中，而不是复制， 你可以为 markdownpicpicker.exe 添加上 -linkonly 参数。在 markdownpicpicker.exe 所在目录打开 cmd, 输入:

markdownpicpicker.exe -linkonly

建议大家使用 AutoHotKey 来启动程序。这样可以把整个流程缩短到 2 秒钟。AutoHotKey 的配置示例：

#n::Run D:\Program Files\MarkdownPicPicker_v1.0.0\MarkdownPicPicker_v1.0.0\MarkdownPicPicker.exe -linkonly

需要首先使用 QQ 截图或者其他截图工具将图片保存到剪贴板中，然后按下设定好的快捷键即可。Markdown 格式的图片链接就已经保存到剪贴板中了。在需要使用的地方直接粘贴。
