# жҢүдҪҚжҲ–иҝҗз®— |

| иҝҗз®—иЎЁзӨәе°ҶдёӨдёӘж“ҚдҪңж•°зҡ„дәҢиҝӣеҲ¶з ҒиҝӣиЎҢжҢүдҪҚжҲ–иҝҗз®—пјҢ дёҖзңҹеҲҷзңҹ гҖӮ
дҫӢеҰӮпјҡ

  • жҲ‘д»¬и®Ўз®— 1|2 е°ұжҳҜи®Ўз®— 001|010пјҢз»“жһңдёәпјҡ011 д№ҹе°ұжҳҜеҚҒиҝӣеҲ¶ 3гҖӮ
  • и®Ўз®— 2|3 е°ұжҳҜи®Ўз®— 010|011пјҢз»“жһңдёәпјҡ011 д№ҹжҳҜ 3гҖӮ
  • и®Ўз®— 2|4 е°ұжҳҜи®Ўз®— 010|100пјҢз»“жһңдёәпјҡ110 еҚҒиҝӣеҲ¶дёә 6

йҖҡиҝҮдёҠйқўдҪ еә”иҜҘеҸҜд»ҘеҸ‘зҺ°пјҢеҪ“жҲ‘д»¬е°Ҷ2дёӘ2зҡ„ж¬Ўж–№ж•°иҝӣиЎҢ|иҝҗз®—пјҢе…¶е®һе°ұжҳҜе°Ҷ2иҖ…зү№жңүзҡ„ж Үеҝ—дҪҚ1 ж•ҙеҗҲеҲ°дёҖиө·гҖӮ иҝҷжҳҜ 2 зҡ„ж¬Ўж–№ж•°жүҖзү№жңүзҡ„пјҢдҪҝз”ЁиҝҷдёҖзү№жҖ§жҲ‘д»¬е°ұеҸҜд»ҘдҪҝз”ЁдәҢиҝӣеҲ¶дҪҚжқҘеӯҳеӮЁй…ҚзҪ®гҖӮ

# жҢүдҪҚжҲ–иҝҗз®— &

& иҝҗз®—иЎЁзӨәе°ҶдёӨдёӘж“ҚдҪңз¬Ұзҡ„дәҢиҝӣеҲ¶иҝӣиЎҢжҢүдҪҚдёҺиҝҗз®—пјҢ еҗҢзңҹжүҚдёәзңҹпјҢдёҖеҒҮеҲҷеҒҮ
дҫӢеҰӮпјҡ

  • жҲ‘д»¬и®Ўз®— 1&2 е°ұжҳҜи®Ўз®— 001&010пјҢз»“жһңдёәпјҡ 000 д№ҹе°ұжҳҜ 0
  • и®Ўз®— 2&3 е°ұжҳҜи®Ўз®— 010&011, з»“жһңдёәпјҡ 010 д№ҹе°ұжҳҜ 2
  • и®Ўз®— 2&4 е°ұжҳҜи®Ўз®— 010&100пјҢз»“жһңдёәпјҡ000 д№ҹе°ұжҳҜ 0

жҲ‘д»¬еҸ‘зҺ°пјҢеҪ“жҲ‘д»¬е°Ҷ2дёӘ2зҡ„ж¬Ўж–№ж•°иҝӣиЎҢ&иҝҗз®—пјҢе…¶е®һе°ұжҳҜе°Ҷ2иҖ…жүҖзү№жңүзҡ„ж Үеҝ—дҪҚ1 е…ЁйғЁзҪ®дёә0
