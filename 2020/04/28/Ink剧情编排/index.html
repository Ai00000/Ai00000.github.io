# 快速食用

  1. 下载 inky 编辑器
  2. 编写 ink 文件
  3. 下载安装 unity 支持
  4. 找到 ink 示例，将对应的 ink 生成的 json 导入运行游戏即可。

# unity 使用流程

在 unity 中，我们需要通过代码来驱动我们的剧情的进行。

  1. 首先我们需要 创建至少2种Prefab ，一种表示文字，一种表示按钮。

  2. 然后我们需要创建一个 story 对象，载入对应的 json 文件
    story = new Story (inkJSONAsset.text);

  3. 判断 story 能否继续，若能继续使用代码创建对应的游戏对象

  1. 要记得每次显示时清除上一次的显示结果

# 书写规则

ink 有一套自己的书写格式，不符合规范时无法编译。

  • 选项 + xx 在选择后会重复 选项
  • 选项 + [xx] 在选择后不会重复 选项
  • …
