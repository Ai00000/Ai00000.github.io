# AB 包


  • LZ4 比标准压缩程度小，但对于内存的占用相对较小，在使用时无需全部解压（推荐使用）
  • ETI 在资源包中不包含资源的类型信息
  • FR 重新打包时需要重新构建包和 ClearFolders 不同，它不会删除不存在的包。
  • ITTC 增量构建检查时，忽略类型数的更改
  • Append Hash 将文件哈希值附加到资源包名上
  • SM 严格模式，如果打包出错，则打包直接失败
  • DRB 运行时构建
