# 前言

🔹ScrollView 是我们经常使用的一个 UI 组件，但时常容易忘记具体的使用细节，故在此文予以记录。主要用于需要滑动的区域

# 组成部分

ScrollView 包含 Content、Scrollbar、OneValueChange 三部分组成。

  1. Content 用于显示内容，通过 mask 遮罩来过滤大于显示区域的内容。
  2. Scrollbar 用于滚动条、
  3. OneValueChange 用于滚动监听
